# EngineeringAdmissionCutoffLearningSystem

### Required Modules
----------------
The following open source programs need to be installed on your computer using pip or anaconda:
	pip3 install -U scikit-learn scipy matplotlib
	pip3 install pandas
	pip3 install numpy
	pip3 install tensorflow==2.0
	pip3 install keras


Once the required modules have been successfully installed, execute the src/run.py file.
